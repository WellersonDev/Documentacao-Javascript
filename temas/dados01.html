Variáveis

Variáveis são usadas para armazenar dados, dessa forma, é possível depois referênciar esses dados atráves do nome que foi dado para variável.

console.log('Teclado');
console.log(20.00);
 // Armazenando o nome e valor do produto dentro de variáveis.
var nome = 'Teclado';
var preco = 20.00;
console.log(nome);
console.log(preco);
